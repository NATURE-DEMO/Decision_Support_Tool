# Specific Site DST (legacy)

!!! warning "Legacy application"
    The Specific Site DST (`pages/2_Specific_Site_DST_v2.py`) is a standalone page that predates the current Integrated DST. Its functionality — pre-configured demo site viewing, Level 1–3 assessment tabs, and GitHub-based content loading — is fully provided by the **[Integrated DST](integrated_dst.md)** under the **Specific Site** mode, with additional features including authentication, multi-expert consensus submission, and editable assessment tables.

    The legacy page remains accessible at `/Specific_Site_DST_v2` but is no longer the primary deliverable.

For documentation of the current Specific Site workflow see **[Integrated DST — Mode 1: Specific Site DST](integrated_dst.md#mode-1-specific-site-dst)**.

## Demo sites

Six pre-configured site configurations are available, covering five NATURE-DEMO demonstration sites:

| Site | Location | Primary CI / hazard focus |
|------|---------|--------------------------|
| Demo 1-A | Lattenbach Valley, Austria | Torrent control / railway — debris flows, extreme rainfall, slope instability |
| Demo 1-B | Brunntal, Austria | Alpine roads and paths — slope stability, snowmelt floods |
| Demo 2 | Brasov City, Romania | Urban green infrastructure — heat island, drought, air pollution |
| Demo 3 | Slovenia | River / flood protection — fluvial flooding, river erosion |
| Demo 4 | Zvolen, Slovakia | Road and railway crossroads — wildfire, flooding, extreme weather |
| Demo 5 | Globocica, North Macedonia | Hydropower dams — drought, sedimentation, water resource stress |
